## 📝 技术细节

### CSS 变量系统
所有颜色都使用 CSS 变量定义，确保主题一致性：

```css
:root {
  --background: #f0f9ff;    /* 天蓝色背景 */
  --accent: #0ea5e9;        /* 天蓝色强调 */
  --accent-2: #0284c7;      /* 深蓝色强调 */
  --card-2: #e0f2fe;        /* 浅蓝色卡片 */
}
```

### 组件自动适配
由于使用了 CSS 变量，所有组件自动适配新主题：
- `CardLink` - 卡片链接组件
- `Nav` - 导航栏组件
- `Footer` - 页脚组件
- 所有页面组件
